Проект: Music Journal
Описание проекта

Music Journal — это веб-приложение для публикации музыкальных постов. Пользователь может зарегистрироваться, войти в систему, создавать посты и просматривать посты других пользователей.

Цель проекта

Создать простой сайт-блог, где можно публиковать музыкальные записи и хранить их в базе данных.

Основные функции
регистрация пользователя
авторизация пользователя
выход из аккаунта
добавление постов
просмотр всех постов
удаление постов
API для получения постов в формате JSON
Используемые технологии
Python (Flask)
HTML (Jinja2 шаблоны)
Bootstrap (для оформления)
SQLAlchemy (работа с базой данных SQLite)
Flask-WTF (формы)
Git (контроль версий)
Структура проекта
app.py — основной файл приложения
data/ — модели базы данных
forms/ — формы регистрации, входа и постов
templates/ — HTML страницы
db/music.db — база данных SQLite
Как запустить проект
Установить зависимости:
pip install -r requirements.txt
Запустить сервер:
python app.py
Открыть в браузере:
http://127.0.0.1:8080
Вывод

Проект реализует базовый функционал веб-блога и демонстрирует работу с базой данных, авторизацией и веб-интерфейсом.
